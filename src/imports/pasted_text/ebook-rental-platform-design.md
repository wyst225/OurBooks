Design a complete responsive frontend UI for a modern, elegant electronic book rental platform. The website should have a calm, premium “old money” aesthetic with a reading-focused atmosphere. The visual style must be minimal, clean, and sophisticated, using soft warm white, butter and beige tones as the primary colors, complemented with dark brown typography and subtle muted gold accents. The design should feel luxurious yet simple, comfortable for long reading sessions, with generous spacing, soft shadows, rounded corners and highly readable typography. Use an elegant serif style for headings and a modern clean sans-serif for body text. The overall feeling should be calm, premium, intellectual and book-focused.

The design must include a complete responsive layout system for desktop, tablet and mobile devices. Every page must adapt properly across screen sizes, with carefully considered breakpoints and fluid layouts. Special attention must be given to the reading interface so that it works exceptionally well on mobile phones and tablets, providing a comfortable and distraction-free reading experience. Navigation patterns should adapt for smaller screens, using collapsible menus, stacked layouts and touch-friendly interactions. All components must scale responsively, including book cards, navigation elements, filters, forms and modals.

Create the public facing website structure that includes a main landing page, a browse books page, a book detail page, an about us page, a contracts and legal documents page, and authentication interfaces. The landing page should introduce the platform with a refined hero section, a featured book area, a section explaining the benefits of the service, and a curated selection of featured or popular books. The browse books page should allow users to search and explore the digital library using filters such as genre, author, popularity and new releases. Books should be displayed in a clean card-based layout that works well across desktop and mobile devices.

The book detail page should present the book cover, title, author, description and rental information, with options to preview the book, add it to the cart, or rent it. Logged-in users should be able to preview the first page of every book through a preview feature. The preview interface should open a simple, elegant reading view that displays the first page in a comfortable layout.

Include authentication interfaces for login and sign up, designed as minimal, centered forms with clean input fields and clear actions. After logging in, users should have access to a personal account area accessible through a dropdown menu in the navigation bar. This menu should contain links to the user’s books, rental history, account settings, support chat and logout.

Create a user dashboard page where logged-in users can see their rented books, the remaining rental time for each book and access the reading interface. The reading interface should be designed as a distraction-free digital reader optimized for long reading sessions, with excellent responsiveness for phones and tablets. The reader should support clear typography, comfortable margins, page navigation controls and reading progress indicators. On mobile devices the reading interface should prioritize readability, full-screen content and intuitive touch navigation.

Include an e-commerce style flow with a cart page and a checkout page. The cart page should display selected books, allow removal of items and show a clear order summary. The checkout page should contain billing information fields, payment method options and a final confirmation step to complete the rental.

Add a floating support chat feature that allows logged-in users to contact customer service. The chat interface should open in a compact panel and allow users to send messages easily. This chat system will later connect to an admin interface for responses.

Design an admin access flow where administrators can log in using a special email and password. After logging in, they should see an admin dashboard interface structured with a sidebar navigation. The admin area should allow management of books, preview pages, users and support chats. The books section should support adding, editing and removing electronic books as well as controlling which pages are available in preview mode. The user management section should allow viewing and managing user accounts. The support chat section should allow administrators to read and respond to user messages.

The design system should include reusable components such as navigation bars, book cards, buttons, form inputs, dropdown menus, modals, tables, chat interfaces and reading components. All elements must follow consistent spacing rules, typographic hierarchy and color usage. The layout should use a structured grid system and scalable spacing so that the design remains clean and balanced across all device sizes.

The final result should be a complete responsive frontend design system and page set for an elegant electronic book rental platform that includes public pages, user authentication, a personal reading dashboard, an optimized reading interface, e-commerce rental flow, support chat functionality and a full administrative management interface, all designed with a calm premium aesthetic that prioritizes readability and user comfort across desktop, tablet and mobile devices.
